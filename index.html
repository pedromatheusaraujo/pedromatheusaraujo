Selected Projects

Automatic Guitar Tuner (Closed-Loop DC Motor Control)

Embedded controls • DSP frequency estimation • Real-time actuation

Developed a closed-loop tuning system that estimates pitch from vibration sensing and drives a DC motor to correct string tension.

  • Control logic converts pitch error (measured in cents)→ motor command with stable convergence on real hardware.
  • Handled nonlinearities and noisy measurements through iterative PI tuning, testing, and filtering.
  • Gained experience designing and integrating PCBs, control systems, WiFi, analog sensors, and 3D-printed components

Automated Gypsum Scraper-Blade System (Industrial Controls)

PLC + DCS synchronization • Sensors + pneumatics • Live plant deployment

Automated seam detection and blade avoidance on a conveyor belt scraper system to prevent belt damage and improve product recovery.

  • Integrated inductive sensing, pneumatic actuation via solenoid control, and manual override controls.
  • Added PLC logic through the DCS line to synchronize timing with belt speed and optimize performance.
  • Enabled recovery of approximately $16,800 worth of gypsum product per day with system installed on two belts.
  • Redlined legacy documentation to reflect updated system schematics and wiring diagrams, maintaining accurate version control.
  • Resources (Videos & Schematics)

Time-of-Flight (ToF) People Counter (ESP32)

Embedded sensing • Edge-case handling • Wi-Fi data link

ESP32-based occupancy tracker mounted above a door frame to count real-time entries and exits, including edge cases such as partial entries and simultaneous crossings.

  • Detection leveraged a ToF sensor, with characteristic signal patterns used to distinguish entries from exits.”
  • Occupancy data was transmitted wirelessly via Wi-Fi to a secondary ESP32 for monitoring and logging.
  • Designed 3D-printed enclosure + electronics integration (sensor, PCB, status LEDs).
